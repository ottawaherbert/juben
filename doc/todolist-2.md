# AI Studio - 核心优化任务清单 (Phase 2)

## 核心目标：引入“全局创作视点 (Creative Vision)”与 AI 风格化引擎

在完成了基础的工业化流程后，第二阶段的核心是赋予作品“灵魂”。我们将引入导演、编剧、对标作品的风格设定，并让这些设定深度影响后续的剧本生成、分镜拆解和画面渲染。

### 任务列表

- [x] **Task 1: 修复与优化大纲结构 (Structure)**
  - [x] 在 `Structure` 页面的节拍模板中增加“自由创作 (无模板)”选项，解除强制选择。
  - [x] 优化 UI 文案，区分“分集宏观三幕”与“单集微观节拍”，减少认知混淆。

- [x] **Task 2: 建立“全局创作视点” (Global Creative Vision)**
  - [x] 在 `Inception` (起点) 页面新增“创作视点 / 导演阐述”模块。
  - [x] 包含以下维度：
    - **类型基调 (Genre)**：悬疑、科幻、武侠、爱情等。
    - **视觉与镜头风格 (Visual Style / Director)**：模仿特定导演（如王家卫的抽帧与霓虹、韦斯·安德森的对称与高饱和）或特定美学（如赛博朋克）。
    - **叙事与台词风格 (Narrative Style / Writer)**：模仿特定编剧（如阿伦·索尔金的密集快节奏台词、昆汀的黑色幽默）。
    - **对标作品 (Reference Works)**：具体的电影/电视（如《银翼杀手2049》+《盗梦空间》）。

- [x] **Task 3: 升级 AI Prompt 引擎 (注入灵魂)**
  - [x] 改造 `Script` (剧本) 页面的 AI 生成逻辑：在生成台词和动作时，自动读取“叙事与台词风格”和“类型基调”。
  - [x] 改造 `Storyboard` (分镜) 页面的 AI 拆解逻辑：在生成镜头语言（景别、运镜）时，自动读取“视觉与镜头风格”（例如：如果选了迈克尔·贝，AI 会自动增加更多 Low Angle 和 Dynamic Pan）。

- [x] **Task 4: 资产与画面的风格化统一 (Stylized Rendering)**
  - [x] 在 `Assets` (资产) 页面生成角色/场景参考图时，将“视觉风格”和“对标作品”作为 Midjourney/Stable Diffusion 级别的 Style Prompt 自动追加，确保全片视觉高度统一。
