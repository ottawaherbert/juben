# Phase 3 / Phase 4 待办事项 (To-Do List)

## 1. 多维度创作视点生成器 (Multi-Dimensional Creative Vision Generator)

**功能描述：**
在“起点 (Home)”页面输入灵感种子并点击“一键立项”后，系统不再只生成单一的创作视点，而是提供多个不同风格的“项目包装”版本供创作者选择。这能极大激发创作者的灵感碰撞，探索同一个故事内核在不同视听语言下的可能性。

**核心配置项：**
*   **版本数量 (Version Count)：** 用户可自定义生成的版本数量（1 到 10 个）。
*   **国家/地区风格 (Region/Country Style)：** 用户可选择特定的国家或地区电影工业风格。
    *   *注意：此选项仅影响“Creative Vision 创作视点”（如对标导演、视觉风格、色彩、节奏等），不改变故事的核心 Logline 和人物设定。*

**支持的国家/地区及代表性风格（包含系统精选的成熟电影市场）：**
1.  **中国大陆 (China)：** 第五代导演的浓烈色彩与宏大叙事（如张艺谋）、第六代导演的社会写实（如贾樟柯）。
2.  **中国香港 (Hong Kong)：** 霓虹灯与抽帧的浪漫（王家卫）、暴力美学与双雄对决（吴宇森）、宿命论与黑色犯罪（杜琪峰）。
3.  **美国 (United States)：** 好莱坞爆米花商业大片（迈克尔·贝）、独立电影/心理惊悚（A24制片厂）、黑色幽默与多线叙事（昆汀·塔伦蒂诺）。
4.  **日本 (Japan)：** 细腻温情的家庭伦理（是枝裕和）、唯美青春（岩井俊二）、赛博朋克与哲学动画（押井守/大友克洋）、武士史诗（黑泽明）。
5.  **韩国 (South Korea)：** 极致的阶级隐喻与类型融合（奉俊昊）、暗黑复仇美学（朴赞郁）、极度写实的犯罪惊悚（罗泓轸）。
6.  **法国 (France)：** 新浪潮的自由跳跃剪辑（戈达尔）、极致的浪漫与神经质（让-皮埃尔·热内）、高概念商业动作（吕克·贝松）。
7.  **意大利 (Italy)：** 新现实主义的粗粝感、超现实与梦境交织（费里尼）、通心粉西部片（赛尔乔·莱昂内）。
8.  **印度 (India)：** 宝莱坞色彩绚丽的歌舞史诗、深刻探讨社会现实的佳作（阿米尔·汗）。
9.  **英国 (United Kingdom)：** 快速剪辑与英式黑色幽默（盖·里奇）、严谨的非线性叙事与冷峻色调（克里斯托弗·诺兰）、古典庄园时代剧。
10. **西班牙 (Spain)：** 浓烈高饱和色彩与性别探讨（阿莫多瓦）、极致的悬疑反转与密闭空间（奥里奥尔·保罗）。
11. **伊朗 (Iran)：** 极简主义与诗意长镜头（阿巴斯）、密集的对白与中产阶级道德困境（阿斯哈·法哈蒂）。

**生成示例（以“程序员死亡倒计时”故事为例）：**
*   **版本 A（美国 - 好莱坞商业风）：** 动作/科幻，迈克尔·贝风格，快节奏，高对比度，爆炸与追车。
*   **版本 B（美国 - 独立电影风）：** 心理惊悚，A24 风格，慢热压抑，冷色调，强调主角的精神崩溃。
*   **版本 C（美国 - 黑色幽默风）：** 喜剧/悬疑，昆汀风格，荒诞，碎嘴台词，非线性叙事。
*   **版本 D（中国香港 - 银河映像风）：** 犯罪/悬疑，杜琪峰风格，高反差打光，宿命感，雨夜与霓虹灯。
*   **版本 E（日本 - 赛博朋克风）：** 科幻/哲学，押井守风格，冷峻蓝绿色调，探讨生命与数据的边界。

**实现路径：**
1. 在 Home 页面“一键立项”按钮旁增加配置面板（Slider 选择 1-10 个版本，Dropdown 选择国家/地区，默认“全球随机混合”）。
2. 调整 AI Prompt，使其根据选择的地区和数量，返回一个包含多个 Creative Vision 对象的 JSON 数组。
3. 在界面上以横向卡片（Carousel）的形式展示生成的多个版本，用户点击“应用此视点”后，再将其写入全局状态的 `currentProject.creativeVision` 中。
