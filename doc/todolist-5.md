# O.M.N.I. 系统进阶优化与专业化重构 (To-Do List 5)

## 第一阶段：交互与易用性优化 (细节打磨)
- [x] **全局进度与状态指示**：在顶部导航栏增加“项目健康度”或“待办事项”提示（例如：“您有 3 个新角色尚未生成参考图”）。
- [x] **错误边界与重试机制**：AI 接口调用失败时，提供优雅的错误提示（如 Toast）和一键重试按钮，而不是让页面卡死或弹出原生 alert。

## 第二阶段：摄影棚 (Studio) 的高级时间轴与合成
- [x] **多轨时间轴 (Multi-track Timeline)**：引入真正的可视化时间轴组件（类似 Premiere 或剪映的底部轨道），允许用户拖拽调整视频片段、配音、BGM、音效的相对位置。
- [x] **云端合成导出**：实现真正的视频合成（可能需要借助 FFmpeg.wasm 在前端合成，或发送到后端处理），允许用户将当前集数的所有分镜导出为一个完整的 MP4 文件。

## 第三阶段：剧本 (Script) 的精细化控制
- [x] **节点式剧本编辑器**：将剧本拆分为更细的“节”或“动作”，允许用户针对单句台词或单个动作进行 AI 重写或扩写。
- [x] **情绪与镜头语言标注**：在剧本阶段就允许用户（或 AI 自动）标注每句台词的情绪（影响 TTS 语气）和建议的镜头景别（影响分镜生成）。
